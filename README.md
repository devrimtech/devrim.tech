## ðŸ›  Installation & Set Up

1. Install pnpm

   ```sh
   curl -fsSL https://get.pnpm.io/install.sh | sh -
   ```

2. Install dependencies

   ```sh
   pnpm i
   ```

3. Create SSL files (optional)

   ```sh
   pnpm run cert
   ```

4. View SSL build (optional)

   ```sh
   pnpm run ssl
   ```

5. View unsecure build

   ```sh
   pnpm run dev
   ```

6. Build fully

   ```sh
   pnpm run build
   ```
