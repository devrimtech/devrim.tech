Intro

I started programming ever since I was 8. I started with a Minecraft modding course and a few game hacking courses and grew out of there. My first experience with a language that I learned on my own was processing. After I learned processing I went on to game development using Unity and C#. I grew into C# more than Unity so I started to learn that. I was learning a lot as I grew older, so naturally The first hackathon I went to, I learned how to use website development languages like HTML/CSS and JavaScript. I have branched off into other languages like Processing and Python.

I have got into the Linux ecosystem. Although my knowledge of hardware is limited, I can understand most problems with both software and hardware.

Work

My first projects include making a mod for Minecraft and some console programs in C#. I made a calculator in the C# console and a game. While at an eBay event I programmed a PigLatin translator in both Python and C#. The bot for the Discord server that me and my friend run is made by me in JavaScript using the Discord.js library. The bot's purpose is to help users with a lot of functionality and for the ScribeHacks hackathon that me and my friend are organizing.

Projects that I have created with friends at hackathons include a rating website called RateMate that was made at the hackathon, Los Altos 4. While at a VMWare highschool program, my team created a website called GreenPoints that generates challenges that help the user be more green. While I was at OmniHacks, my team created a game in Unity called Lost, about the reality of Depression. At Solar Hacks I created a program called VoiceInput, with my team in Python that turned the user's speech into computer input. At SigmaHacks, my team created a live stock viewer for the terminal called StockPye.

About

Outside of computers, my hobbies include reading, art, and writing. I like to read mostly non-fiction books, my favorite books include the Mystery of Shakespeare series, Hatchet, and the Giver.

In terms of art, I like to draw different characters. I usually draw using a pencil and paper in school, but at home I use a XP-Pen Deco 01. My passion for art came from the art everyday that I challenged myself to do over the summer of 2019.

For writing, I have written some short stories when I was younger and have participated in the NaNoWriMo event in 2019. My work currently includes the story Camp Atom.
